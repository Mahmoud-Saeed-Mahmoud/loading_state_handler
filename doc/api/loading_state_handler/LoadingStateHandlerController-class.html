LoadingStateHandlerController

LoadingStateHandlerController class

A controller for LoadingStateHandlerWidget.

This controller is used to control the state of the LoadingStateHandlerWidget. It is not necessary to use a StatefulWidget or trigger a rebuild when changing the state of the widget. The controller provides a simple way to set the state of the widget.

The controller can be used to set the state of the widget to CurrentStateEnum.normal, CurrentStateEnum.loading, CurrentStateEnum.data, CurrentStateEnum.empty, or CurrentStateEnum.error.

The LoadingStateHandlerWidget takes a controller property which is an instance of LoadingStateHandlerController. The controller is used to control the state of the widget.

When the state of the widget is changed, the LoadingStateHandlerWidget will rebuild with the new state.
